# Code of Conduct

This project uses a simple contributor covenant: be respectful, constructive, and specific.

## Expected behavior

- Assume good intent while reviewing ideas and code.
- Keep feedback about the work, not the person.
- Make space for beginners and non-native English speakers.
- Prefer actionable suggestions over vague criticism.

## Unacceptable behavior

- Harassment, threats, hate speech, or personal attacks.
- Publishing private information.
- Repeated off-topic or disruptive comments.

Maintainers may moderate issues, pull requests, and discussions to keep the project useful and welcoming.
