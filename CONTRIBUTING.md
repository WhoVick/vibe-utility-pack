# Contributing

Thanks for helping improve Vibe Utility Pack.

## Good first contributions

- Add sample inputs for one of the ten tools.
- Improve accessibility labels or keyboard behavior.
- Add parser edge cases for finance CSV, changelog messages, or meeting notes.
- Improve documentation with screenshots and short workflows.

## Local setup

```bash
npm test
npm start
```

The app is dependency-free. Keep changes focused, readable, and easy to review.

## Pull request checklist

- The change is scoped to one tool or one shared behavior.
- `npm test` passes.
- User-facing text is clear and concise.
- New parser behavior includes an example in `examples/` or README notes.

## Maintainer workflow

Maintainers triage issues by tool area, label bugs/features/docs, and review pull requests for:

- Security and privacy impact.
- Browser compatibility.
- Accessibility basics.
- Clear examples and documentation.
